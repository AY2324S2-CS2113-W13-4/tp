To get started, you can type 'help' to see a list of commands!

_______________________________________________________________________________

These are the current available features and the format that you need
to follow to use it in using this software:

1. To add books
   add BOOK_TITLE (optional)/quantity NUMBER_OF_COPIES

2. To delete current existing book
   delete INDEX (optional)/quantity NUMBER_OF_COPIES

3. To mark book as borrowed by book index
   borrow INDEX /by USER_NAME

4. To mark book as borrowed by book title
   borrow BOOK_TITLE /by USER_NAME

5. To extend the due date of a borrowed book by a week by book index
   extend INDEX /by USER_NAME

6. To extend the due date of a borrowed book by a week by book title
   extend BOOK_TITLE /by USER_NAME

7. To return a book by book index
   return INDEX /by USER_NAME

8. To return a book by book title
   return BOOK_TITLE /by USER_NAME

9. To list all the books added
   list /sortby default

10. To list all the books added in alphabetical order
   list /sortby alphabetical

11. To list all the users and their borrowed books
   list /sortby user

12. To edit the title of the existing book by index
   edit INDEX /title NEW_BOOK_TITLE

13. To edit the title of the existing book by current book title
   edit CURRENT_BOOK_TITLE /title NEW_BOOK_TITLE

14.To find a specific user and their borrowed books:
   find /by user USER_NAME

15.To find a book in the inventory
   find /by book BOOK_TITLE
