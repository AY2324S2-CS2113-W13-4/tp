Welcome to BookMarked, a one-stop app for all your librarian needs!
To get started, you can type 'help' to see a list of commands!

_______________________________________________________________________________
_______________________________________________________________________________


_______________________________________________________________________________

These are the current available features and the format that you need
to follow to use it in using this software:
1. To add books
   add BOOK_TITLE (optional)/quantity NUMBER_OF_COPIES
2. To delete current existing book
   delete NUMBER_ACCORDING_TO_LIST
3. To mark book as borrowed
   borrow BOOK_TITLE by USER_NAME
4. To extend the due date of a borrowed book by a week:
   extend BOOK_TITLE
5. To unmark book as returned
   return BOOK_TITLE
6. To list all the books added
   list /sortby default
7. To list all the books added in alphabetical order
   list /sortby alphabetical
8. To list all the borrowed books by return date
   list /sortby returndate
9. To list all the users and their borrowed books
   list /sortby user
10. To edit the title of the existing book
   edit NUMBER_ACCORDING_TO_LIST /title DESCRIPTION
   example: edit 1 /title book
11.To find a specific user and their borrowed books:
   find /by user USER_NAME
12.To find a book in the inventory
   find /by book BOOK_TITLE
